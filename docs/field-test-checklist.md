# Field Test Checklist

- Confirm Guide and Journey open in airplane mode without permission prompts.
- Start a hike and verify foreground permission is requested once.
- Compare each battery mode’s update cadence and thermal impact.
- Press Stop while GPS and a mock request are active; verify immediate pause.
- Kill and relaunch the app; verify it remains paused.
- Confirm guide/journal remain usable while paused.
- Export JSON and inspect journeys, entries, completion, settings, and outbox.
- Test large text, VoiceOver/TalkBack, sunlight contrast, and one-handed/gloved taps.
- Record device, OS, weather, duration, battery delta, and anomalies.
