# Product Brief

MacKaye Fieldbook is a local-first field guide and private hiking journal. Guide and journal functions work without an account or network. GPS and synchronization occur only after explicit user actions.

Phase 1 proves the app shell, fictional offline content, SQLite journal, foreground tracking boundary, activity stop control, manual mock sync, export, diagnostics, and safety language. Real maps, trail data, accounts, sharing, moderation, and production sync are intentionally deferred.

Success means a user can browse fictional content, maintain a local journey, start a foreground hike, stop every active capability immediately, restart without auto-resume, and export their records.
